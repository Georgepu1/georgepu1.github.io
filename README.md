# George Pu Personal Website

Personal research website for [georgepu1.github.io](https://georgepu1.github.io/).

## Development

This site is a Create React App project. Use Node 22 locally.

```bash
nvm use 22
npm ci --legacy-peer-deps
npm start
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
nvm use 22
npm run build
```

## Deploy

The repo is configured for GitHub Pages. After reviewing changes:

```bash
npm run deploy
```
